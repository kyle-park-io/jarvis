# CLI (apps/cli) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `apps/cli` — the user-facing frontend. `jarvis today` runs the daily pipeline and prints the plan; `jarvis plan --date=…` does it for a date; `jarvis alerts` prints the alerts. This completes the MVP: a person can run `jarvis today` and get their plan.

**Architecture:** A dependency-injected `runCli(argv, deps)` function holds all the dispatch logic and is fully unit-testable (temp data root + real folder connector + a capturing output). A pure `formatAlerts()` renders alerts for the terminal. A thin `bin.ts` wires the real dependencies (`resolveDataRoot`, `folderConnector`, today's date, `process.stdout`). Command parsing uses Node 22's built-in `util.parseArgs` — no CLI-framework dependency.

**Tech Stack:** TypeScript (ES2022, strict), pnpm workspaces, vitest, tsx (to run the TS entrypoint).

## Global Constraints

- **`apps/cli` depends on `@jarvis/core`, `@jarvis/store`, `@jarvis/connectors`, `@jarvis/scheduler`** (workspace). No CLI-framework dependency — use `node:util`'s `parseArgs`.
- **All dispatch logic lives in `runCli(argv, deps)`** with injected dependencies (`dataRoot`, `connectors`, `today`, `out`), so it is testable without touching the real filesystem/clock/stdout. `bin.ts` only wires the real deps.
- **MVP run path is `tsx`** (`pnpm jarvis today`). A single-binary bundle is deferred to a distribution follow-up.
- **Node ≥ 22.** Language: English for code, comments, identifiers, commit messages. Conventional Commits.
- **No `Co-Authored-By: Claude` / no "Generated with" trailers** in commits.
- **TDD:** every logic function gets a failing test first. Filesystem tests use a per-test temp directory (`fs.mkdtempSync`) and clean it up.

```ts
import type { Alert } from '@jarvis/core';

export function formatAlerts(alerts: Alert[]): string {
  if (alerts.length === 0) return 'No alerts.\n';
  return alerts.map((a) => `[${a.severity}] ${a.type}: ${a.message}`).join('\n') + '\n';
}
```

```ts
import fs from 'node:fs';
import { parseArgs } from 'node:util';
import { runDailyPlan } from '@jarvis/scheduler';
import type { Connector } from '@jarvis/connectors';
import { formatAlerts } from './render';

export interface CliDeps {
  dataRoot: string;
  connectors: Connector[];
  today: string;
  out: (text: string) => void;
}

const HELP = `jarvis — personal task planner

Usage:
  jarvis today             Show today's plan
  jarvis plan [--date=D]   Show the plan for a date (default: today)
  jarvis alerts            Show today's alerts
  jarvis help              Show this help
`;

export async function runCli(argv: string[], deps: CliDeps): Promise<number> {
  const command = argv[0];
  switch (command) {
    case 'today':
      return showPlan(deps, deps.today);
    case 'plan': {
      const { values } = parseArgs({
        args: argv.slice(1),
        options: { date: { type: 'string' } },
        allowPositionals: true,
      });
      return showPlan(deps, values.date ?? deps.today);
    }
    case 'alerts': {
      const result = await runDailyPlan({ dataRoot: deps.dataRoot, connectors: deps.connectors, date: deps.today });
      deps.out(formatAlerts(result.alerts));
      return 0;
    }
    case undefined:
    case 'help':
      deps.out(HELP);
      return 0;
    default:
      deps.out(`Unknown command: ${command}\n\n${HELP}`);
      return 1;
  }
}

async function showPlan(deps: CliDeps, date: string): Promise<number> {
  const result = await runDailyPlan({ dataRoot: deps.dataRoot, connectors: deps.connectors, date });
  deps.out(fs.readFileSync(result.planPath, 'utf8'));
  return 0;
}
```

- [ ] **Step 1: Write the entrypoint `bin.ts`**

`apps/cli/src/bin.ts`:
```ts
#!/usr/bin/env node
import path from 'node:path';
import { toISODate } from '@jarvis/core';
import { resolveDataRoot } from '@jarvis/store';
import { folderConnector } from '@jarvis/connectors';
import { runCli } from './cli';

const dataRoot = resolveDataRoot();

runCli(process.argv.slice(2), {
  dataRoot,
  connectors: [folderConnector(path.join(dataRoot, 'streams'))],
  today: toISODate(new Date()),
  out: (text) => process.stdout.write(text),
})
  .then((code) => process.exit(code))
  .catch((error: unknown) => {
    process.stderr.write(`Error: ${error instanceof Error ? error.message : String(error)}\n`);
    process.exit(1);
  });
```
