# CLI Time Logging (`jarvis log`) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `jarvis log <stream> <hours> [--date=D]` command that records worked hours (via the store's existing `addTimeLog`). This completes the engine's self-correcting pace: logged hours reduce a stream's `remainingWeekly`, so the next `jarvis today` demands less of a stream you've already worked.

**Architecture:** One new `log` case in the existing `runCli` dispatch (`apps/cli/src/cli.ts`). It parses the stream id + hours (positionals) and an optional `--date`, opens the store DB, calls `addTimeLog`, and confirms. No new package, no engine change — `@jarvis/store` already exports `addTimeLog`.

**Tech Stack:** TypeScript (ES2022, strict), pnpm workspaces, vitest.

## Global Constraints

- **Change is confined to `apps/cli`** — `runCli` gains a `log` command. It uses `openDb` + `addTimeLog` from `@jarvis/store` (already exported). No changes to core/store/scheduler/connectors.
- **`log` semantics:** `jarvis log <stream> <hours> [--date=YYYY-MM-DD]`. Both positionals required (else usage + exit 1); `hours` must be a finite number > 0 (else error + exit 1); date defaults to `deps.today`. On success: `addTimeLog(db, { date, streamId, hours })`, print `Logged <h>h to <stream> on <date>.`, exit 0.
- **Node ≥ 22.** Language: English. Conventional Commits. No `Co-Authored-By: Claude` / "Generated with" trailers.
- **TDD:** failing test first. Tests use a temp data root + real folder connector + a capturing `out`.

(b) Add a line to the `HELP` template (after the `alerts` line):

```
  jarvis log <stream> <hours> [--date=D]   Log hours worked on a stream
```

(c) Add the `log` case to the `switch (command)` (before `case undefined:`):
```ts
    case 'log': {
      const { values, positionals } = parseArgs({
        args: argv.slice(1),
        options: { date: { type: 'string' } },
        allowPositionals: true,
      });
      const streamId = positionals[0];
      const hoursArg = positionals[1];
      if (streamId === undefined || hoursArg === undefined) {
        deps.out('Usage: jarvis log <stream> <hours> [--date=YYYY-MM-DD]\n');
        return 1;
      }
      const hours = Number(hoursArg);
      if (!Number.isFinite(hours) || hours <= 0) {
        deps.out(`Invalid hours: ${hoursArg}\n`);
        return 1;
      }
      const date = values.date ?? deps.today;
      const db = openDb(deps.dataRoot);
      try {
        addTimeLog(db, { date, streamId, hours });
      } finally {
        db.close();
      }
      deps.out(`Logged ${hours}h to ${streamId} on ${date}.\n`);
      return 0;
    }
```
